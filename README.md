## Quick Start

1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio-website.git
cd portfolio-website
```

2. Open `index.html` in your browser or use a local server
```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have live-server installed)
npx live-server
```

3. Customize the content in `index.html` to match your information

## Customization

### Personal Information
- Update the name, title, and description in the hero section
- Replace placeholder project information with your actual projects
- Update contact links with your social media profiles
- Modify the skills section to reflect your expertise

### Styling
- Colors can be changed in `styles.css` (look for CSS custom properties)
- Fonts can be updated by changing the Google Fonts import
- Layout modifications can be made in the respective CSS sections

### Adding Projects
To add a new project, copy the project card structure in the HTML and update:
- Project title and description
- Technology tags
- Live demo and GitHub links
- Project image (replace placeholder)

## Deployment

### Netlify Deployment
1. Push your code to GitHub
2. Connect your GitHub repository to Netlify
3. Set build settings (if needed)
4. Deploy automatically on every push

### Custom Domain
1. Purchase your domain from a registrar
2. In Netlify, go to Domain settings
3. Add your custom domain
4. Update DNS settings with your registrar
